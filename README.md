# Patient-Comorbidity-Risk-Assessment-Analysis
An interactive health data science project analyse chronic conditions and patient comorbidity profiles. It helps to visualize multi-condition risk groups and compare Which Chronic Disease carries the highest average mortality risk. Includes the CSV dataset and the .ipynb Jupyter Notebook file.

## 1. 📌Project Overview
The goal of this project is to analyze patient health records data. Using Python and data analysis libraries the project investigates the relationships between various health indicators—such as Diabetes, Chronic Kidney Disease (CKD), Hypertension, Liver Disease, and Heart Attack Risk—to uncover meaningful medical insights.

## 2. Tools Used
•	Python

•	Pandas & NumPy

•	Plotly Express

•	Seaborn & Matplotlib

## 3. Dataset
Source: Healthcare patient records dataset. Data Contains: 20 Columns and 10000 Rows covering chronic conditions. The Column names are Age, Blood_Pressure, Diabetes, Chronic_Kidney_Disease (CKD), Hypertension, Liver_Disease, Heart_Attack_Risk.

## 4. Steps Followed
•	Cleaned the healthcare dataset using Pandas to handle missing values and structure data formats.

•	I replaced missing values with appropriate baseline statistics(mean,mode)

•	Created a 'BMI Categorization' column.

•	Did visualization using cleaned data.

## 5. Key Insights
. Clear patterns emerge showing that lifestyle choices (like physical activity levels) significantly alter critical risk bars for heart attacks. 

. Patients who already have conditions like Diabetes, Hypertension, or Liver Disease have a much higher average death risk (Mortality Risk Percentage) than healthy individuals. 

. Patients who have Diabetes are much more likely to also have Chronic Kidney Disease.

## 8. Conclusion
The final notebook is a clear and colourful tool. It helps anyone quickly see how deeply connected a patient's lifestyle and daily habits are to serious health risks. Old age, high blood pressure, diabetes, and heavy drinking are the biggest factors leading to dangerous health conditions and higher mortality rates.
